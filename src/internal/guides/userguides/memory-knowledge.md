# 记忆与知识

EverEvo 有两套互补的「知识」能力：**记忆系统**（记住关于你和工作的事实）与**知识库 RAG**（检索你上传的文档）。

## 记忆系统

记忆系统跨会话持久化，分三层：

- **核心层（user_facts）**：身份 / 偏好 / 约束，永久保存，可手动锁定。召回时强制注入。
- **情节层（memory_items）**：对话中抽取的事实，按指数衰减（重要性低遗忘更快），超龄且低分时被 TTL 清理。
- **时序知识图谱**：实体 + 关系（bi-temporal），支持多跳查询与可视化（力导向 / 层次布局）。

**核心隔离原则**：身份永久，TTL / 衰减只动情节 —— 「90 天后会忘了我吗？不会。」

## 它怎么记住你

- 每轮存你的问题向量；每隔几轮，用一个便宜模型异步抽取稳定事实与图谱实体 / 关系。
- 召回时按语义相似度取 Top-K 注入系统提示，让模型「想起」相关上下文。
- 同一事实会自动去重（精确 + 语义），不会因为你重复说就存 5 遍。

> 想换记忆绑定的句向量模型？在设置里切换，已有记忆会自动重新向量化迁移。

## 知识库（RAG）

独立的文档检索：

1. **添加**：上传文本 / 文档 → 自动切片 + 向量化。
2. **搜索**：输入查询 → 返回最相关的片段。
3. **浏览**：翻看已入库的内容。

知识库与记忆分开管理，互不干扰。两者都需要一个句向量模型（市场下载 `all-MiniLM-L6-v2` 即可启用）。

## 在哪里看

- 记忆：**知识库页顶部的「对话记忆」**区块（条目数 / 预览 / 清空）与知识图谱可视化。
- 知识库：**知识库**的三 Tab 界面。
