# 工具箱

工具箱是 EverEvo 的核心：按**模型类型**提供专门的推理工具。每个工具的输入 / 输出都符合该类型的语义，而不是把向量 / 概率塞进一个通用文本框。

## 为什么按类型？

AI 模型的输入 / 输出形状差异巨大：

- 句向量模型 → 两段文本 → 相似度分数（或句库 → Top-K）
- 图像分类模型 → 一张图 → 类别 + 置信度
- 目标检测模型 → 一张图 → 框 + 类别

用一个「文本框」硬套所有模型并不实用。工具箱为每类模型定制交互。

## 当前支持

| 类型 | 工具 | 状态 |
|------|------|------|
| 句向量（sentence-transformers） | 语义相似度 / 语义搜索 | ✅ 可用 |
| 图像分类 | 选图 → 类别 + 置信度 | 🚧 规划中 |
| 文本分类 / 目标检测 | — | 📋 计划 |

## 使用流程

1. 在市场下载一个对应类型的模型（例如句向量下载 `all-MiniLM-L6-v2`）。
2. 打开工具箱，选对应工具（如「语义相似度」）。
3. 载入模型 —— 工具箱会**自动探测模型类型**（读 metadata / `config.json` / I/O 形状），若类型不符会给明确提示。
4. 按工具的输入方式使用：
   - **语义相似度**：输入两段文本 → 输出 0~1 的相似度分数。
   - **语义搜索**：维护一个句库 → 输入查询 → 返回 Top-K 最相似的句子。

## 语义相似度原理（句向量）

载入句向量模型后，工具会：分词 → 模型前向 → 用 attention mask 加权 mean-pool 得到句向量 → 两向量余弦相似度即为分数。整个过程本地完成。
